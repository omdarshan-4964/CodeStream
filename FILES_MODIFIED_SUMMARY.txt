- **What Changed:** Fixed Next.js 16 params Promise handling

**Before:**
```typescript

) {

```

**After:**
```typescript
export async function POST(
  req: Request,

  const resolvedParams = await params;
  // ...
  const { roomId } = resolvedParams;
```

**Why:** Next.js 16 makes route params a Promise; must be awaited
